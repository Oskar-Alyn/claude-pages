Life from one table

Each dot pulls some colors closer, and pushes others away.

Before the rule

Each dot is one particle

Every dot up there is a single particle, and each one has a color. Think of the colors as teams, just a few of them, with every particle on a team behaving the same way.

A particle can't see the whole picture. It only reacts to the neighbors right around it, following one rule.

The one rule

Every color has a feeling about every other

Pick any two colors. A single number says how the first affects the second: pull closer, or push away, and how strongly. Collect a number for every pair and you have a table. That table is the whole rulebook.

This is the actual table running the swarm up top.

pulls closer pushes away

Read it as: the row's color feels this way about the column's color.

Every number in the table works the same way. The simplest case is one color with a single number.

Push apart, they spread out
Pull together, they clump

What falls out · 1

It only comes alive in a crowd

A rule needs something to act on. Set a handful of particles loose and almost nothing happens. Pour in a crowd and the same table starts growing cells, clusters, and moving shapes.

A handful, nothing takes shape
A crowd, structure appears

What falls out · 2

The feeling can be one-sided

The pull doesn't have to run both ways. Pink can chase gold while gold flees pink. Neither one ever wins, so they keep moving, and you get roving packs and chains.

Both attract, they settle into a blob
One chases the other, they never stop

What falls out · 3

Change one number, change the world

One changed number is all it takes. Below are two worlds whose tables are identical except for the highlighted cell. That single value is the difference between a quiet drift and tight, restless clumps.

One cell set to push, they keep apart
That one cell flipped to pull, they bind together
